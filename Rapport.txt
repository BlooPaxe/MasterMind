-) La bibliothèque conio.h sera utilisée dans la procédure "saisie_code"

-) On a définie une structure de donnée "rangée" contenant un tableau et un entier représentant sa taille

-) Une procédure init_jeu qui permet de saisir le nombre de tentatives, le nombre de couleur, ainsi que la taille de la rangée, qui affiche aussi les couleurs pouvant être séléctionnées

-) Une procédure init_tab qui remet les valeurs du tableau à la chaine vide

-) Une procédure saisie_joueurs qui permet de saisir le pseudo des deux joueurs, le j1 étant le premier à saisir le code

-) Une procédure saisie_code (Maintenant incluse dans la procédure de jeu ) :
 			~) La boucle while s'arrete dès que la taille du code est égale à la taille des rangées
			~) La fonction getch() permet de ne pas afficher la saisie dans la console, et donc le code est caché

-) Une fonction booléenne verif qui vérifie si la réponse saisie est égale au code secret

-) La procédure jeu qui permet de lancer le jeu : 
		~) La boucle while s'arrete des que le compteur "tour" est strictement supérieur au nombre de tentatives imposé
		~) Le deuxieme joueur saisie la réponse
		~) La procédure s'arrete si la fonction verif envoie "true", et affiche le vainqueur
		~) Si la reponse est incorrecte, le premier joueur saisie l'indice qui permet d'indiquer au deuxieme joueur ses fautes
		~) Quand on sort de la boucle while, cela signifie que le nombre de tentatives maximum est atteint, et donc le deuxieme joueur a perdu

-) Pour le main, on fait appel à toute les procédures précédentes pour initialiser le jeu. Ensuite, on utilise une boucle while pour que le jeu se répète jusqu'a atteindre le nombre de manche saisi par le joueur.
Dans la boucle while, on fait appel à la procédure de jeu 2 fois, une fois pour le premier joueur, et une fois pour le deuxieme, ensuite on garde le nom du gagnant de la manche à partir de la procédure jeu.
On ajoute 1 au score du gagnant de la manche.
Pour finir, on affiche le gagnant de la partie, avec le score le plus élevé, ou on affiche une égalité.
